RoBoat

A Wifi Controlled MotorBoat

A ROS operated motorboat. The package lets you control the rotation of the prop and the motor speed with the keyboard. It also lets you visualize gyro data in RVIZ, aswellas displaying ultrasonic sensor data as a cone. The arduino code prints the gyro data and ultrasonic data through serial for the package to read and then display in RVIZ.

Project members: Jiwoo Jang, Richard Li.

Personal contribution: All the software.

  • Manufacturing still in Progress
